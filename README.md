# モグラたたきゲーム（Web）

HTML・CSS・JavaScriptを使用して制作した、ブラウザで遊べるモグラたたきゲームです。  
JavaScriptによるイベント処理、ランダム表示、タイマー制御などの基礎的なWeb技術を用いて実装しました。

## 概要
制限時間内にランダムで表示されるモグラをクリックし、スコアを競うゲームです。  
シンプルな構成ですが、ユーザー操作と画面更新の流れを意識して開発しました。

## 使用技術
- HTML  
- CSS  
- JavaScript  

## 操作方法
- 表示されたモグラをクリックするとスコアが加算されます  
- 制限時間が終了するとゲーム終了となります  
- 「再スタート」ボタンで再度プレイできます
